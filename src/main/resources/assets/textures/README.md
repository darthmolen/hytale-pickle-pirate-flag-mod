# Pickle Pirate Flag Textures

This folder should contain the following texture files. Create these using an image editor like GIMP, Photoshop, or Aseprite.

## Required Textures

### pickle_flag.png (32x32 pixels)
The main flag texture. Minimalist pickle pirate design:
- Green pickle shape (vertical oval)
- Simple pirate bandana on top (red or black)
- X marks for eyes (skull-like aesthetic)
- Transparent background

### PickleFlag_Marker.png (16x16 pixels)
Map marker icon for discovered flags:
- Small green flag icon
- Recognizable pickle shape
- Bright/visible colors

### PickleFlag_Marker_Undiscovered.png (16x16 pixels)
Map marker icon for undiscovered flags:
- Same design as discovered but grayed out
- Or use a question mark icon
- Lower contrast/muted colors

### flagpole.png (16x64 or similar)
The wooden pole texture:
- Brown wood grain
- Simple design

## Design Tips

1. Keep it simple - Hytale uses a stylized aesthetic
2. Use 32px or 64px per block unit for textures
3. Test visibility at different zoom levels
4. Ensure good contrast for map markers

## Tools

- [Aseprite](https://www.aseprite.org/) - Pixel art editor
- [GIMP](https://www.gimp.org/) - Free image editor
- [Blockbench](https://blockbench.net/) - For 3D models with textures
