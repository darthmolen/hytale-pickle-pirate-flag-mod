# Block Textures

Place your textures here:

## Required Files

### pickle_flag.png (32x32 or 64x64)
The flag texture with pickle pirate design:
- Green pickle silhouette
- Pirate bandana/hat
- X marks for eyes
- Transparent or colored background

### flagpole.png (16x16 or 32x32)
Wood texture for the pole:
- Brown wood grain
- Simple repeating pattern

## Creating Textures

1. Use any image editor (GIMP, Photoshop, Aseprite)
2. Create at 32x32 or 64x64 pixels
3. Save as PNG with transparency
4. Keep colors bold and readable at small sizes
